## 二、Frontmatter 验证

### 2.1 必需字段
- [ ] `name` 字段存在
- [ ] `name` 与目录名一致
- [ ] `description` 字段存在
- [ ] `description` 遵循公式：功能描述 + 执行方式 + 触发场景

### 2.2 参考型 Skill 特定验证
- [ ] 无 `disable-model-invocation` 字段（或为 false）
- [ ] `allowed-tools` 只包含只读工具（Read、Grep、Glob）
- [ ] 无 `user-invocable` 字段（或为 false）

### 2.3 任务型 Skill 特定验证
- [ ] `disable-model-invocation: true` 已设置
- [ ] `user-invocable: true` 已设置
- [ ] `allowed-tools` 包含必要的读写工具
- [ ] 权限配置遵循最小权限原则

### 2.4 可选字段
- [ ] `allowed-tools` 配置合理
- [ ] 无冗余配置

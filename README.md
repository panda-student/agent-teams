# Agent Teams

多 Agent 协作技能，用于创建多个 Claude Code 实例进行团队协作。

## 安装

```bash
# 克隆到你的 skills 目录
git clone git@github.com:panda-student/agent-teams.git ~/.openclaw/workspace/skills/agent-teams
```

## 使用模式

1. **开发需求模式** - 适用于新功能开发、需求实现
2. **修复bug模式** - 适用于问题排查、bug 修复
